## Badge Color Reference

| Color | Hex | Use For |
|-------|-----|---------|
| Success | `success` | Passing, Ready, Active |
| Warning | `yellow` | Preview, Beta, Pending |
| Critical | `critical` | Error, Failed |
| Blue | `blue` | Information, Version |
| Green | `green` | Features, Support |
| Orange | `orange` | Features, Languages |
| Purple | `blueviolet` | Patterns, Architecture |
